# Aluminum Profile Go-Kart（铝型材小型卡丁车）

一个基于铝型材的小型 DIY 卡丁车 / 移动平台项目，面向学习、交流与个人实践，**非商业用途开源**。

---

## 📌 项目简介

本项目是一个 **小尺寸铝型材结构的卡丁车 / 小车平台**，最初设计目标是能够放入 **大众 ID.3** 的后备箱中，方便运输与收纳。

项目特点：

- 尺寸相对紧凑，偏向学习与实验用途  
- 适合机械结构实践、DIY 装配与兴趣研究  
- **非商品、非量产设计、非工程认证方案**

由于不同车型、使用场景、材料选择以及个人经验水平存在差异，**本项目中的尺寸与结构并不保证适用于所有情况**，使用者需自行评估并调整。

---
